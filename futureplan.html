관심기술 공부


대학교 2학년 전공 과목에 자바 프로그래밍 과목을 수강하면서 웹 페이지를 제작하는 과제를 수행한 적이 있습니다.
팀 프로젝트를 수행하며 웹 페이지 제작에 관심을 가지게 되었고, 웹 페이지 제작뿐만 아니라 백엔드까지 수행하는 풀스택 개발자를 꿈꾸게 되었습니다.
또한, 현재 웹 프로그래밍을 수강하며 웹 페이지 제작에 대한 기초적인 지식을 학습 중입니다.

위는 제가 대학 졸업 전에 이룰 목표들입니다.

정보처리기사에 응시하려면 4학년 1학기 이상이어야 가능하므로, 그전까지는 웹 공부에 치중할 생각입니다.
현재까지 통틀어 보았을 때 제가 가장 부족한 면은 학점이므로 전공 공부에 집중하여 학점을 올리려 노력하고 있습니다.
4학년 학기 시작 이후에는 실무에 필요한 교육인 풀스택 국비 프로그램에 지원하여 풀스택 개발 교육을 목표로 하고 있습니다.
